# ENEM Corporation Website

A polished single-page website for ENEM Corporation, an import-export company specializing in chemical raw materials and industrial minerals.

## Features

- Full single-page layout with Home, About Us, Products, and Contact sections
- Fixed top navigation with smooth anchor scrolling
- Animated globe display using `ScrollGlobe` and `Globe` UI components
- Tailwind CSS styling with branded color palette
- Responsive layout for desktop and mobile
- Contact cards and submission form design

## Project structure

- `src/App.tsx` — main marketing website layout
- `src/components/ui` — reusable UI primitives and globe animation components
- `src/lib/utils.ts` — shared utility helpers
- `src/index.css` — Tailwind CSS setup and global styles
- `tailwind.config.cjs` & `postcss.config.cjs` — Tailwind configuration

## Scripts

- `npm install` — install project dependencies
- `npm run dev` — start the local development server
- `npm run build` — build the production bundle
- `npm run preview` — preview the production build locally
- `npm run lint` — run ESLint checks

## Getting started

```bash
npm install
npm run dev
```

Visit the URL shown in the terminal to explore the ENEM Corporation landing page.
